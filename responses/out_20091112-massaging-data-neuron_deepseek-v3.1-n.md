# Massaging the Data for Neurontin? (Nov 2009)

## 1. SUMMARY

这篇2009年的评论文章讨论了一项发表于《新英格兰医学杂志》的研究,该研究揭露了Neurontin(加巴喷丁,gabapentin)临床试验数据中的系统性操纵问题。研究人员比较了Parke-Davis及后续辉瑞公司的内部试验报告与公开发表的文献,发现了选择性报告的有力证据。

关键发现包括:20项临床试验中超过一半(11项)未以完整研究论文形式发表;在发表的9项试验中,有7项报告的统计学显著主要终点与原始试验方案中定义的不同;几乎所有从内部文件到公开发表的变化都"导向了医学文献中更有利的呈现"。作者指出这种做法在统计学上是严重不当的——在查看数据后更改主要终点会破坏随机对照试验的科学完整性。

虽然文章指出其他机构(包括加拿大卫生研究院资助的试验)也存在类似的选择性报告问题,但这篇NEJM论文揭示的模式特别令人不安,因为这些研究似乎被用于推动加巴喷丁标签外用途的推广。

## 2. HISTORY

**法律后果扩大:**
- 2004年辉瑞已因Neurontin市场营销问题支付4.3亿美元和解
- 2008-2014年间辉瑞因加巴喷丁相关的标签外营销指控支付累计超过20亿美元罚款

**证据基础转变:**
- 2011年后FDA开始要求新药申请时同步提交所有临床试验方案和结果
- 2014年临床试验注册网站(ClinicalTrials.gov)报告要求大大加强
- 2017年FDA进一步收紧上市后研究透明度要求

**Neurontin实际临床表现:**
- 加巴喷丁和普瑞巴林在神经病理性疼痛中确实建立了有效循证基础
- 带状疱疹后神经痛、糖尿病神经病变成为标准治疗选项
- 但上述系统性操纵损害了患者和医生对制药行业的信任

**神经病理性疼痛治疗领域发展:**
- 多模式镇痛理念确立,加巴喷丁/普瑞巴林成为一线用药
- 2010年后SNRI类药物(度洛西汀)在部分适应症中显示不劣效性
- 2015年后CGRP单抗偏头痛预防药获批改变了该领域格局

## 3. PREDICTIONS

- **选择性报告问题的普遍性判断正确** ✓
  - 后续研究(E.g. 2018年BMJ系统综述)确实显示约30-40%已发表试验的主要终点与注册方案不一致
  - 问题未局限于制药行业资助的研究,公共资助试验同样存在

- **该丑闻对制药行业长期声誉的影响被低估** ✗
  - 作者认为"我们行业不应该让这种事情首先发生"
  - 实际情况是后续十年间药企不当行为揭露案例持续增加(如2012年GSK、2013年J&J诉讼)
  - 系统性透明化改革效果有限,公众信任恢复缓慢

- **对加巴喷丁临床价值的判断过于悲观** PS
  - 文章暗示研究方法偏见可能意味着药物实际效果存疑
  - 但后续大规模真实世界研究和meta分析确认了加巴喷丁在神经病理性疼痛中的疗效
  - 操纵数据的行为可耻,但未完全否定该药的临床价值

- **监管改善的推进速度被低估** ✗
  - 作者未预料到FDAAA 2007和2016年21世纪治愈法案带来的透明度改革
  - 试验注册、结果报告、数据共享成为强制性要求,远比2009年更严格
